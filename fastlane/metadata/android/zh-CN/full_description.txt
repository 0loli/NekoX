Nekogram 在官方 Telegram 安卓客户端的基础上主要增加了以下功能:

• 一键复读
• 一键收藏
• 在群聊中忽略屏蔽的用户的发言
• 无引用消息转发
• 通过 ID 创建 @
• 允许非管理员用户查看群聊权限和管理员
• 为普通会话选择地图预览提供商
• 通过消息菜单快速提升和限制群组成员
• 通过消息菜单快速查看用户历史消息
• 删除单个缓存的文件
• 自定义贴纸显示大小
• 显示和导出消息详情
• 无限的收藏贴纸
• 按用户、群组、频道、机器人、管理员或可通知分类会话
• 支持至多 8 个账号
• 支持使用机器人账号登录
• 允许首次登录时不同步通讯录
• 显示 ID 和数据中心
• 全透明的状态栏
• 更改姓名顺序
• 在界面上隐藏自己的手机号码
• 内置中文和日文语言包
• 使用系统字体和 emoji
• 存储缓存到应用私有目录下
• 隐藏代理赞助商频道
• 设置在所有设备上显示敏感媒体
